# Bungie Marathon 2026 Concept Tracker

This is a concept tracker website for a video game called Marathon released in 2026 by Bungie.

A tracker website utilizes an API that allows users to make requests for information related to the game, such as player profiles, item information, etc.

As of now, Bungie has not released an API for the game, for now all so far information is statically created by hand.

The website features multiple tabs in which the user can peruse through and utilize their respective features.
